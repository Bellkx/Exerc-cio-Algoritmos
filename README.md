# Cadastro de Alunos em C/C++

Este projeto consiste em um programa desenvolvido em linguagem C/C++ que permite o cadastro de até 5 alunos, armazenando seus nomes e três notas individuais.

## Funcionalidades
Cadastro de até 5 alunos
Armazenamento de:
Nome do aluno
Três notas
Cálculo automático da média de cada aluno
Exibição de uma tabela contendo:
Nome
Notas
Média
Situação (Aprovado ou Reprovado)
Identificação do aluno com a maior média da turma
## Lógica aplicada

O programa utiliza conceitos fundamentais de programação estruturada:

Vetores (arrays) para armazenar os dados dos alunos
Função para calcular a média das notas
Estruturas de repetição (for ou while) para entrada e exibição dos dados
Estrutura condicional (if/else) para determinar a situação do aluno:
Média ≥ 7 → Aprovado
Média < 7 → Reprovado
## Exemplo de saída

O programa exibe uma tabela organizada com todas as informações dos alunos, além de destacar qual aluno obteve a maior média da turma.

## Objetivo

Este exercício tem como objetivo reforçar conceitos básicos de programação, como manipulação de vetores, uso de funções, estruturas de controle e organização de dados.
