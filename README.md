# Exerc-cio-Algoritmos
Cadastro de Alunos em C/C++  Programa que cadastra até 5 alunos, armazenando nome e três notas. Calcula a média de cada aluno, exibe uma tabela com notas, média e situação (Aprovado ou Reprovado), e identifica o aluno com maior média.  Utiliza vetores, função para cálculo da média, estruturas de repetição e condicionais.
